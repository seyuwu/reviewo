# Opinia marketing agents — shared thread

## Product context (fixed)

- **Opinia** — opinia.ru: opinions, ratings, battles, tops, Chrome extension
- **Opinia Games** — games.opinia.ru / dota.opinia.ru: Dota teammate/party search
- **Launch:** search opens **2026-07-19 19:00 MSK**; waitlist + TG `@opinia_official` / support `t.me/opinia_support`
- **Team:** 2 people; CIS / RU primary; soft launch via TG + Discord + VK, not Instantly-first
- **Positioning Games:** fast convenient party/teammate search
- **Constraint:** empty search kills trust; don't split EN yet

## Task

Marketing for next 2–4 weeks (pre-launch → first week live). Impact vs effort.

## Round 1

### Agent C — Ideas

**10 идей (14 дней)**

1. Пин + пост «открытие 19.07 19:00» в `@opinia_official` — TG канал  
2. Ежедневный короткий пост 17–19.07: боль → поиск → ссылка — TG  
3. ЛС 10–15 друзьям/тиммейтам: профиль + канал — личные сети  
4. 12–15 нативных постов в TG LFG («ищу/собираю пати + профиль») — TG чаты  
5. ЛС админам чатов: «можно анонс в день старта?» — TG DM  
6. 5 постов в Discord #lfg + 3 DM модерам — Discord  
7. 5–8 VK групп Dota/пати — пост или ЛС админу — VK  
8. В 19:05 у вас онлайн 2 пати + 5 соло в поиске (seed) — продукт  
9. Пост «как за 2 минуты создать профиль / найти пати» со скринами — TG  
10. После каждой своей катки — ссылка 1–2 тиммейтам — Dota/Steam chat  

**TOP 5 + метрика**

| Идея | Метрика за 14 дней |
|------|-------------------|
| Друзья → профили | +15–25 профилей |
| TG LFG посты + админы | 10+ касаний, 2+ ответа админов |
| Seed в 19:05 | ≥5 живых карточек в поиске в первый час |
| Канал: пин + ежедневные посты | +подписки, клики на dota.opinia.ru |
| Discord/VK зеркало | +5–10 заходов не из «своих» |

**Seed на день 1 (пустота)**

1. За 1 час до старта: вы + друг + заранее договорённые 4–6 человек включают LFG  
2. 2 готовых пати со слотами (не пустые названия)  
3. Пост в канал в T+0: «поиск открыт, вот кого сейчас видно / зайди»  

**Кросс Opinia ↔ Games**

1. С `opinia.ru` (intent / nav): явный путь «Ищу пати в Dote» → games/dota  
2. В Games после создания профиля: «есть OpinIA — рейтинги/мнения» одной строкой (не давить)

### Agent B — Critic / prioritizer

**Критерии (scorecard)**  
1. Даёт профиль или человека в поиске за ≤48ч?  
2. Делается вами двоими без новых интеграций?  
3. Риск бана / пустого продукта?  
4. Повторяемо на следующей неделе?  
5. Метрика не vanity (не «охваты», а профили/пати)?  

**Kill list (ловушки)**  
1. «Автоматизации Cursor / multi-agent cold mail» — не приводит дотеров  
2. Instantly AI prospects — не ваш ICP  
3. Reddit / Product Hunt на этой неделе — мимо + бан-риск  
4. MMR filter / polish UI вместо людей в поиске — оптимизация пустоты  
5. «Построить свой Discord сервер» — пустая комната, нет distribution  

**Must-do 72ч до запуска**  
1. Пин в TG + описание канала  
2. Свои профили заполнены  
3. 8–12 друзей: профиль + подписка  
4. Таблица касаний; ≥10 TG/Discord касаний  
5. Договорённость seed-группы на 19:00  
6. Смоук: мобилка + OG ссылки  
7. Текст поста на 19:00 готов  
8. Кто жмёт `searchLive` — один ответственный  

**Must-do 7 дней после**  
1. Каждый вечер: вы в поиске / пати  
2. 1 пост/день в канал (как пользоваться / кого ищем)  
3. Добивать LFG-чаты, где был ответ  
4. Баги блокеры — сразу  
5. Считать: профили / заходы в поиск / собранные пати  
6. 3–5 ЛС админам на коллаб-анонс  
7. Не распыляться на EN/Instantly  
8. Раз в 2 дня: что сработало → повторить  

**Harsh truth про US > RU в GA**  
16 vs 9 — шум + VPN/боты/превью. Это не спрос США. Решайте по профилям и пати из СНГ.  

**3 ставки на 2 недели**  
1. **Ручной soft launch (TG+Discord+друзья)** — единственный путь к первым 30 профилям; сигнал: касания в таблице + профили  
2. **Seed в момент открытия** — без этого поиск мёртвый; сигнал: ≥5 online в первый час  
3. **Канал как хаб** — все ссылки ведут сюда; сигнал: подписки + клики в день старта
